Hello World

Aug 20, 2016

I was thinking of writing a blog and i choose hexo for it.

Why Hexo?


I wanted to make a static blog like jeykll and wanted to host it in Github.(Cos why not)

Since writing in markdown is pretty easy i decided to go with it rather than the traditional wordpress which i had used .
But this is an argument that ended long time ago. Markdowncheatsheet.

I didnt choose jeykll mostly since its too much in the market. So i decided to go with something new .I was actually planning to use it if this fails.But it worked brilliantly.

Hexo is in node.js making it really fast and light. Even the documentation was so easy to pick up.
The ease in Deployment is something i found beautiful. I did face a little bit of probelm finding themes. Cos there are a lot to try.
But some errors where there like everyother one,but i was able to solve all of them with ease.

It is so much better than wordpress as in ease of use after getting a hang of it.

Give it a shot and try it out.
